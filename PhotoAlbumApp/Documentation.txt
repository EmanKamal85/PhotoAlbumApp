This is an application designed and implemented specifically as a database photostorage, providing the user a way to manage, organize, modify and browse
photos with the ability to modify existing database items and a description for each photo.
The app uses Room database as a data provider, provides permission request from the user to access the Android device storage, reduces the size of selected photo while
displayed in the UI and utilizes ViewModel technology to provide LiveData interface with the database while using the app.
The project is implemented in Java language.
